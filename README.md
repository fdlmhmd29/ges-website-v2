# Environmental Consultant Website

This project is built with **SvelteKit**. It provides a landing page for an environmental consulting company, a blog with admin management, and a simple client carousel.

## Features
- Landing page with rotating client names
- Blog listing and individual post pages
- Simple admin login to create and delete posts
- Posts stored in `src/posts.json`

## Development

```bash
npm install
npm run dev -- --open
```

## Building

```bash
npm run build
```
